# WordPress Website Setup (Vagrant + Ubuntu)

## 📌 Project Overview
This project demonstrates how to set up a WordPress website using Vagrant, Ubuntu, Apache, MySQL, and PHP.

## ⚙️ Technologies Used
- Vagrant
- VirtualBox
- Ubuntu (focal64)
- Apache (Web Server)
- MySQL (Database)
- PHP
- WordPress

## 🚀 Setup Steps
1. Created Vagrant environment
2. Configured VM (CPU, Memory)
3. Set up private network
4. Installed Apache, MySQL, PHP
5. Downloaded and configured WordPress
6. Deployed website on Apache

## 🌐 Access
After setup, access the website via: http://192.168.56.26/wp-admin/


## 📂 Project Structure
- Vagrantfile → VM configuration
- WordPress files → Website code

## 🔐 Security Note
Sensitive files like `wp-config.php` are excluded using `.gitignore`.
